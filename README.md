   <h1>Mark株式会社(架空のコーポレートサイト)</h1>
    <h2>コンセプト</h2>
    <p>アニメーションは少なめ、ミニマルかつわかりやすいサイト</p>
    <h2>使用言語</h2>
    <ul>
        <li>HTML5</li>
        <li>CSS3(Sass)</li>
        <li>Javascript</li>
        <li>Vue.js 2</li>
    </ul>
    <h2>アピールポイント</h2>
    <p>IntersectionObserver APIを使用してスクロールアニメーション処理。Newsページ、ContactページはVue.jsを使用して開発。
    Newsページは種別に絞り込み可能。Contactページのバリデーションは非同期処理で実装。</p>
